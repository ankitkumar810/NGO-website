# 🌍 Open Arms Foundation

A modern and responsive NGO website built using HTML, CSS, and JavaScript.  
This project is designed to simulate a real-world donation and volunteer platform with a clean UI and smooth user experience.

## 📌 About the Project

Open Arms Foundation is a frontend-based NGO website that showcases how a real-world nonprofit platform works.  
It includes sections for mission, vision, donations, volunteer registration, and an impact gallery.

This project is built as part of my learning journey to improve my frontend development and UI design skills.

---

## ✨ Features

- 🏠 Modern landing page with hero section  
- 🎯 Separate Mission and Vision sections  
- 📊 Impact gallery with real images  
- 💳 Donation UI with quick amount selection  
- 🧑‍🤝‍🧑 Volunteer registration form with multiple fields  
- 📱 Fully responsive design for all devices  
- 🎨 Clean, modern, and professional UI
